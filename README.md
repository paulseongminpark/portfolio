## Overview

이 포트폴리오는 다음 질문에서 출발합니다.

> “무엇을 만들 것인가”보다  
> **“어떤 기준으로 판단하게 할 것인가를 어떻게 설계할 것인가”**

건축·미학·데이터/AI 등 서로 다른 영역에서  
일관된 **사고 구조(System-Oriented Thinking)**로 문제를 정의하고,  
이를 **실제로 작동하는 구조와 시스템**으로 구현해 온 작업을 정리했습니다.

감각적·정성적 문제를  
관찰 가능한 변수, 상태(State), 조건 관계로 재정의하고  
의사결정이 가능한 구조로 번역하는 과정을 중심으로 구성되어 있습니다.
